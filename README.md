# Horse Trainer AI

A web application that provides personalized training recommendations for 
horses based on their age and breed.

## Features

- Enter your horse's information (name, age, breed)
- Receive customized training recommendations
- Get insights into exercise type, duration, and intensity

## Local Development

1. Clone the repository
2. Create a virtual environment: `python3 -m venv horse_env`
3. Activate the virtual environment: `source horse_env/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run the application: `python app.py`
6. Open http://127.0.0.1:5000 in your browser

## Deployment

This application is ready to deploy to Heroku or similar platforms.
